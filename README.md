# GHST
- Code language: JavaScript
- Framework: React 17.0.1
- UI Library: Material-UI 4.11.3
- Icons: Material-UI Icons 4.11.2

## Install
    Node >= 8.10 with npm >= 5.6
    Google Cloud SDK https://cloud.google.com/sdk/ (just download it)

## Run app locally
    npm i
    npm start

## Test
    npm test

## Production Deploy
